## Firestore for Datastore
### Features;
- Supports ACID transactions, highly consistent
    - A = Atomicity
    - C = Consistency
    - I = Integrity
    - D = Durability
- Highly scalable
    - Horizontal scaling
- Multi-region: High availability and distaster recovery
- Supports NoSQL
- Serverless, no servers required to be managed
- Real-time sync
